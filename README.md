# YujinMath 오답노트 생성기

이 웹앱은 SAT 모의고사 문제를 기준으로 학생별 오답노트를 자동으로 생성해주는 도구입니다.

## 사용법
1. 문제 이미지가 들어있는 ZIP 파일을 업로드합니다.
   - 폴더 구조 예시:
     mock3.zip
     - M1/
       - 1.jpg
       - 2.jpg
     - M2/
       - 1.jpg
       - 2.jpg

2. 오답 정보 엑셀 파일을 업로드합니다.  
   예시 파일: 예시_오답노트_양식.xlsx

3. 자동으로 워드 문서를 생성하여 다운로드할 수 있습니다.
